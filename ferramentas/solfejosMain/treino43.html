Olá, Pessoas!!
Esta página serve para você estudar o solfejo, ou seja, a leitura da partitura através do canto.
Existe uma melodia que está escrita e ela é companhada de 5 botões.
O primeiro botão vai tocar a tonalidade para você encontrar uma referência.
O segundo botão vai tocar o acompanhamento para você cantar.
Os três últimos botões servem para você estudar se ainda tiver dificuldades.
No terceiro eu canto as notas, no quarto os graus, e no quinto eu faço uma leitura rítmica.
Jogue duro. Abração,
Carlinhos.
